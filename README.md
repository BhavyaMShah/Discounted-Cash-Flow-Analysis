# Discounted-Cash-Flow-Analysis
 A Python-based Discounted Cash Flow (DCF) model that estimates enterprise value using FCFF and WACC.

**Features**

FCFF-based DCF valuation

WACC calculation using CAPM

Excel upload support for fast, reusable inputs

Optional manual input mode

Modular, readable Python code

**Tech Stack**

Python 路 Pandas 路 NumPy 路 Excel 路 Jupyter / Colab

**Use Cases**

Equity valuation 路 Investment analysis 路 Finance coursework 路 Interview case studies

**How to Run**

Clone the repo from GitHub, upload the input Excel file (or enter inputs manually), and run the notebook.
